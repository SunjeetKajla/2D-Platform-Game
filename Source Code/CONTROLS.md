# 2D Parkour Platformer - Controls

## Movement Controls
- **A / Left Arrow**: Move Left
- **D / Right Arrow**: Move Right
- **W / Space**: Jump
- **S**: Slide (when on ground)

## Game Controls
- **ESC**: Pause/Resume Game
- **Mouse**: Navigate Menus

## Parkour Mechanics
- **Wall Jumping**: Jump while touching a wall to perform a wall jump
- **Wall Sliding**: Hold against a wall while falling to slide down slowly
- **Sliding**: Press S while running to slide under obstacles
- **Coyote Time**: Brief window to jump after leaving a platform
- **Jump Buffering**: Jump input is remembered for a short time

## Game Features
- **3 Unique Levels**: Urban rooftops, industrial sites, underground tunnels
- **Collectibles**: Gather all coins in each level to progress
- **Save System**: Progress is automatically saved
- **Statistics**: Track your performance across all levels
- **Sound Effects**: Immersive audio for all actions

## Level Progression
1. **Level 1**: Urban Rooftops (10 coins)
2. **Level 2**: Industrial Sites (10 coins) 
3. **Level 3**: Underground Tunnels (15 coins)

Complete all levels to return to the main menu with your stats updated!
